# FacuLemo/loves-you

**FacuLemo/loves-you** Es un sitio dedicado a mi novia: Se genera un corazón giratorio compuesto por la palabra "Te amo". Por encima de este corazón, se muestra un mensaje central y un subtítulo.

## Uso

Cloná o descargá el repo y abrí `index.html` en cualquier navegador moderno. No requiere servidor ni dependencias.

## Archivos

- `index.html` — estructura
- `styles.css` — estilos y animaciones
- `script.js` — genera las palabras sobre la curva del corazón

## Para personalizar...

- **Mensaje central**: editá el `<h1>`, `<p class="subtitle">` y `<title>` en `index.html`
- **Palabras del corazón**: constante `MESSAGE` en `script.js`
- **Velocidad de giro**: `animation: spin-pause 32s` en `styles.css`
- **Cantidad de palabras**: `MIN_WORDS`, `MAX_WORDS`, `WORDS_PER_PX` en `script.js`
